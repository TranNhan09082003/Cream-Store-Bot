CREAM STORE BOT V12 BIG UPDATE
==============================

Mục tiêu của bản v12:
- FIX gấp lỗi bot không nhận được mã đơn hàng cũ trong database
- Không xóa dữ liệu cũ
- Giữ nguyên DB hiện tại
- Tăng độ chính xác khi match webhook PayOS -> order
- Gộp luôn UI QR mới và UI thanh toán sau /oder

FIX CHÍNH TRONG BẢN NÀY
-----------------------
1. Matcher mới cho mã đơn:
   - exact payment_code
   - exact order_code
   - fallback đổi CR_ <-> VB_
   - fallback đọc mã từ description/reference
   - fallback match digits nếu chỉ có duy nhất 1 ứng viên

2. Webhook PayOS:
   - log rõ hơn
   - trả matched_by
   - tự sync payment_code nếu cột có sẵn

3. Script backfill an toàn:
   - scripts/backfill-payment-codes.js
   - chỉ gán payment_code = order_code cho các đơn đang thiếu
   - không xóa dữ liệu

4. UI:
   - paymentQrUi.js
   - paymentUiV118.js

CÁCH UPDATE AN TOÀN
-------------------
1. Backup source hiện tại.
2. Giải nén ZIP này.
3. Copy đè file vào project hiện tại.
4. Nếu DB của bạn có cột payment_code, chạy một lần:
   node scripts/backfill-payment-codes.js
5. Restart bot:
   npm start
