CREAM STORE BOT V11.7 PAYMENT + UI UPDATE
==========================================

Mục tiêu của bản này:
1. Sửa phần webhook PayOS để bot có thể tự đánh dấu đã thanh toán ngay khi PayOS gửi POST webhook
2. Giữ dashboard chạy chung port với webhook
3. Cập nhật giao diện QR thanh toán theo style mới giống ảnh mẫu
4. Không đụng data/, database schema, hay bắt xóa DB

FILE TRONG BUNDLE
-----------------
- src/utils/paymentQrUi.js
- src/services/paymentWebhookHelpers.js
- src/services/webhookServer.js
- src/services/paymentService.integration-example.js

VÌ SAO ẢNH CỦA BẠN CHƯA TỰ NHẬN TIỀN?
------------------------------------
- Trang /payments/payos/return và /payments/payos/cancel chỉ là page người dùng nhìn thấy sau checkout
- Việc tự xác nhận thanh toán phải đi qua:
  POST /webhooks/payos
- Nếu webhook không map được orderCode -> order trong DB, bot sẽ không cập nhật đơn

BẢN NÀY SỬA GÌ?
---------------
- Webhook server mới sẽ:
  1) nhận payload PayOS
  2) đọc data.orderCode
  3) tìm order theo payment_code hoặc order_code
  4) set payment_status = PAID
  5) nếu status đang PENDING_PAYMENT thì chuyển sang PROCESSING
  6) ghi audit log vào staff_logs nếu bảng có sẵn

LƯU Ý QUAN TRỌNG
----------------
- Bản này chỉ sửa nền tảng webhook và UI QR.
